PURPOSE
The purpose of this project is to analyze customer loan data and understand loan applicant characteristics using Python, SQL, and Power BI. The dashboard provides insights into customer demographics, loan amount, credit score, income, employment type, and loan default status to support data-driven analysis.

KEY FINDINGS
The dataset contains approximately 255K customer loan records.
The average credit score is around 574.
The overall default rate is approximately 12%.
Loan applications are distributed across different education levels.
Average income varies by employment type.
Customers apply for loans for multiple purposes such as Home, Business, Education, Auto, and Other.
